# TinyWrite

Just a little writing tool with markdown shortcuts that saves every change to
local indexeddb.

## Install

Download from [releases](https://github.com/dennis84/tiny-write/releases)

## Build

```
npm install

# Build app
npm run dist
# Ubuntu
snap install app/TinyWrite_0.x.x_amd64.snap --dangerous

# Develop web
npm run web

# Develop app
npm run app
```
